[Installing Protoc, Ubuntu]
    ubuntu: sudo apt install protobuf-compiler
    protoc --version

Install grpc and protoc dependencies for your grpc intended project.
    go get -u google.golang.org/grpc
    go get -u github.com/golang/protobuf/protoc-gen-go

Ensure GOBIN and GOPATH env variables are set in PATH 'protoc v1/pb/blog/blogApp.proto --go_out=plugins=grpc:.' to work.
Use command: 'go env'(will print all the go env variables)
    if GOBIN and GOPATH are not set,
        edit ~/.bashrc and add the following;
            # Go
            export GOROOT=/usr/local/go
            export GOPATH=$HOME/go
            export GOBIN=$GOPATH/bin
            export PATH=$PATH:$GOROOT:$GOPATH:$GOBIN
    then:
        source ~/.bashrc
